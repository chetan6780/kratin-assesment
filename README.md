# Healthify - Kratin Assesment Project

Healthify is a web application for sunita,a 65+ years old woman to help her live healthy and happy life.

### Features

-   Diet Plan
-   Yoga Suggestions
-   BMI Calculator
-   Emergency SMS
-   Emergency Medical Help
